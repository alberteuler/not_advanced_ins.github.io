两种常见的errno：

errno 含义
ECONNREFUSED 目标端口不存在
ETIMEDOUT 连接超时

关闭连接

fd:需要关闭的socket。
close的动作只是在将fd的引用计数-1。只有fd的引用计数为0时才会真正地关闭连接。
如果需要立刻终止连接，那么应该采用shutdown。

其中howto定义了shutdown的行为：

howto的值 含义
SHUT_RD 关闭sockfd读的这一半
SHUT_WR 关闭sockfd写的这一半
SHUT_RDWR 读写全部关闭
