3 处理僵尸进程

在子进程结束运行之后，父进程读取其推出状态之前，那么子进程进入僵尸态。如果父进程结束或者异常终止，但子进程仍然继续运行，那么子进程也会进入僵尸态。

父进程需要跟踪子进程的退出状态。因此，子进程结束运行的时候，进程表项会被一直保留，直到父进程读取了其推出状态为止。这个时候，子进程的资源才会被回收。


wait()会阻塞进程，直到该进程的某个子进程结束运行为止。

4.2 信号量

4.2.1 信号量原语

信号量是一种特殊的变量。它只支持两个操作：P(passeren)和V(vrijgeven)，前者用于进入临界区，后者用于退出临界区。这两个操作的功能如下：

  • Passeren : 信号量 > 0时 -1， < 0 时阻塞进程
  • Vrijgeven: 如果有其他进程因为等待SV而挂起，那么唤醒之，如果没有，则SV ++
